Seven Sins of Quantitative Investing

  1. Survivorship Bias: Ignoring companies that went bankrupt or were delisted along the way, focusing only on the current investment targets.
  2. Look-Ahead Bias: Using information that wasnâ€™t publicly available at the time when the simulated decision-making would have occurred.
  3. Storytelling: Creating a narrative after the fact to justify a random pattern obtained from backtesting.
  4. Data Snooping: Over-optimizing the model by adjusting it using the test data.
  5. Transaction Costs: Accurately knowing the transaction costs is only possible by actually executing trades, making it difficult to simulate them.
  6. Outliers: Creating strategies based on a small number of extreme results that are unlikely to be repeated.
  7. Short Positions: Not considering the need to find a lender when taking a short position on a security.
